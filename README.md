# Probe  
A fast, native REST client built with Rust + egui.  
Designed for developers who want power without the bloat.

Probe is a lightweight alternative to Postman/Insomnia no Electron, no cloud sync, no telemetry.  
Just clean `.http` files, instant startup, and a workflow that stays out of your way.

---

## ✨ Features

- **Native performance** built in Rust, using egui for a responsive, minimal UI  
- **`.http` file support** portable, readable request definitions  
- **Fast startup** no heavy frameworks or background services  
- **Local‑first** no accounts, no sync, no tracking  
- **Developer‑centric workflow** keyboard‑friendly, predictable, distraction‑free  
- **Open source** transparent, hackable, and evolving with the community  

---

## 🚀 Why Probe?

Most REST clients today are:

- Slow to launch  
- Heavy on memory  
- Packed with features you don’t need  
- Built on Electron  
- Trying to upsell you a cloud service  

Probe goes the opposite direction.

It’s a **native tool** for developers who value speed, clarity, and control.  
If you prefer a `.http` file and a clean UI over a subscription‑driven dashboard, Probe is for you.
